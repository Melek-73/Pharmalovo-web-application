# Pharmalovo

Pharmalovo is a digital platform that revolutionizes medication access by connecting pharmacies and customers through real-time stock visibility, online ordering, and secure payments.

## Tech Stack

- **Frontend:** react
- **Backend:** Spring Boot
- **Database:** MySQL/PostgreSQL
- **Security:** JWT Authentication

## Prerequisites

- Node.js 18+ and npm
- Java 17+
- MySQL/PostgreSQL
- Maven
